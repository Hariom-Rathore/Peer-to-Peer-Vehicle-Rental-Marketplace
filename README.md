# 🏠 Airbnb Clone

## 📌 About The Project
A full-stack Airbnb clone website where users can:

✨ Features:
- 🔐 Create accounts
- 👤 Login / Register
- 🏡 Add property listings
- 🖼️ Upload images
- 📅 Book places
- 📱 Responsive design

---

## 🛠️ Tech Stack

| Technology | Usage |
|------------|-------|
| Node.js | Backend Runtime |
| Express.js | Server Framework |
| MongoDB | Database |
| EJS | Templating Engine |
| Bootstrap | Frontend Styling |
